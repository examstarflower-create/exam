# 📂 별바라기 족보 시스템

**별바라기** 학우들을 위한 족보 및 자료 아카이브입니다.

## 📚 자료 업로드/다운로드 가이드

📥 자료 다운로드 가이드

### 방법1. 전체 자료 한 번에 받기 (ZIP 다운로드)

페이지 우측 상단의 초록색 <> Code 버튼을 클릭합니다.

가장 아래에 있는 Download ZIP을 클릭합니다.

다운로드된 압축 파일을 풀어서 사용합니다.

(혹은 Github Desktop을 이용해 원하는 파일에 clone하면 모든 파일에 쉽게 접근할 수 있습니다.)

### 방법2. 특정 파일만 하나씩 받기

원하는 파일명을 클릭해서 들어갑니다.

우측 상단에 있는 Download raw file 아이콘(📥, 화살표 모양)을 클릭하면 해당 파일만 다운로드됩니다.

📤 자료 업로드 가이드

### 간접 업로드

이 저장소는 보안 및 관리를 위해 직접적인 업로드를 제한하고 있습니다.
자료를 공유해주실 학우분들은 아래 두 가지 방법 중 하나를 선택해 주세요.

### 방법 1. Fork & Pull Request (누구나 가능 🌟)

별도의 권한 신청 없이, 내 계정으로 복사해간 뒤 자료를 추가하여 요청을 보내는 방식입니다.

Fork 하기: 이 페이지 우측 상단의 Fork 버튼을 눌러 내 계정으로 레포지토리를 복사해갑니다.

자료 올리기: 내 계정에 생긴 레포지토리에 자료를 업로드합니다. (Github Desktop, 웹 업로드 등 편한 방법 사용)

PR 보내기:

원본 레포지토리(여기)로 돌아옵니다.

상단 메뉴의 Pull requests 탭 클릭 -> New pull request 클릭.

Compare across forks를 누르고, 방금 업로드한 내 레포지토리를 선택합니다.

Create pull request를 누르면 관리자에게 업로드 요청이 전송됩니다.

### 방법 2. Contributor 권한 신청 (정기 업로더 추천 🔑)

지속적으로 자료를 공유하고 싶다면 관리자에게 직접 권한을 요청하세요. 승인 시 바로 업로드가 가능합니다.

관리자(노욱진)에게 카카오톡 보내기: Github id 혹은 본인의 이메일

초대 수락:

관리자가 초대를 보내면, 깃허브 혹은 이메일로 온 Invitation을 수락해 주세요.

업로드:

권한을 얻은 후에는 Github Desktop이나 웹 페이지에서 자유롭게 업로드할 수 있습니다.

### 직접 업로드 (동아리방 컴퓨터로 직접 접속 가능)

### 방법 1. Github Desktop 사용 (초보자 추천 🌟)

1. [Github Desktop](https://desktop.github.com/)을 설치하고 로그인합니다.
2. 이 레포지토리 페이지에서 초록색 `code` 버튼 -> `Open with GitHub Desktop`을 클릭하여 내 컴퓨터로 가져옵니다(Clone).
3. 내 컴퓨터의 해당 폴더에 자료를 복사해 넣습니다.
4. Github Desktop 프로그램으로 돌아오면 변경된 파일이 감지됩니다.
5. 좌측 하단에 **Summary(제목)**를 적고 `Commit to main` 버튼을 누릅니다.
6. 상단의 `Push origin` 버튼을 눌러 업로드를 완료합니다.

### 방법 2. Github 웹 페이지에서 바로 업로드 (설치 불필요)

1. 자료를 올리고 싶은 폴더로 이동합니다.
2. 우측 상단의 `Add file` 버튼을 누르고 `Upload files`를 선택합니다.
3. 준비된 파일을 드래그 앤 드롭(Drag & Drop) 합니다.
4. 아래 **Commit changes** 박스에 어떤 자료인지 설명을 적고, 초록색 `Commit changes` 버튼을 누르면 완료됩니다.

### 방법 3. 명령어를 이용하는 방법 (숙련자용 💻)

터미널(Git Bash) 사용이 익숙하신 분들을 위한 방법입니다.

```bash
# 1. 저장소 가져오기 (최초 1회)
git clone <레포지토리 주소>

# 2. 자료 추가 후 스테이징
git add .

# 3. 커밋 메시지 작성
git commit -m "과목명/자료종류 업데이트"

# 4. 원격 저장소로 업로드
git push origin main

```
